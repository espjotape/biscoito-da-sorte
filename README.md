<h1> Projeto - Biscoito da sorte 🍀 </h1>

# 💻 Sobre o desafio

Neste desafio você deverá criar um jogo chamado Biscoito da Sorte, no qual o usuário, **a partir de um clique ou enter**, abre um biscoito com a sua sorte do dia.
